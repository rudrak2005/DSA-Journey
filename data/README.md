# Data Folder

This folder is used to store the student records generated by the program.

The application saves all student information in the `students.txt` file so that the data remains available even after closing the program.

When the program starts, it reads the saved data from the file. Any new student added, updated, or removed is automatically reflected in the file.

I created this folder to practice file handling in C++ and to make the project more realistic by keeping data persistent between program runs.

**Note:**
It is recommended not to edit the `students.txt` file manually unless required.
