# Source Folder

This folder contains the implementation files of the project.

### Files

* **`main.cpp`** → Starts the program and handles the menu-driven interface.
* **`Book.cpp`** → Implements all functions related to the `Book` class.
* **`Library.cpp`** → Implements the `Library` class and manages all book operations.

### What I Learned

While working on these source files, I learned how to separate class declarations from implementations, organize code into multiple files, and build a simple menu-driven C++ application.
