#  Library Management System (C++)

This is a simple **Library Management System** developed in **C++** as a practice project to improve my understanding of **Object-Oriented Programming (OOP)**, **STL**, and basic project structure.

The project provides a menu-driven console application where users can manage books by adding, searching, issuing, and returning them.

---

##  What This Project Can Do

* Add a new book
* Display all books
* Search a book using Book ID
* Issue a book
* Return a book
* Check the availability status of books
* Store multiple books using STL Vector

##  How to Run

Compile the project:

```bash
g++ src/*.cpp -Iinclude -o main
```

Run the executable:

**Linux / macOS**

```bash
./main
```

**Windows**

```bash
.\main.exe
```
