# Source Folder

This folder contains the main implementation of the project.

Files:

* `main.cpp` → Menu and user interaction
* `Person.cpp` → Person class implementation
* `Student.cpp` → Student class implementation
* `StudentManager.cpp` → Student management logic

One thing I learned during this project is that even a small spelling mistake in file names or function names can cause compilation errors. Debugging those mistakes helped me improve my understanding of C++ project structure.
