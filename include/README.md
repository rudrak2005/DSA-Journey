# Include Folder

This folder contains all the header files of the project.

I separated the classes into different header files to keep the code organized and easier to understand.

Files in this folder:

* `Person.h` → Base class
* `Student.h` → Student-related data and functions
* `StudentManager.h` → Handles all student operations

While working on this project, I learned how header files help in managing larger C++ programs.
